# 4-1. 관리자 프론트엔드 대시보드 구축

## 작업 목적
FastAPI 백엔드 이벤트를 운영자가 웹에서 즉시 확인할 수 있도록 `/admin` 기반 관리자 UI를 추가했다.

## 구현 범위
- 관리자 페이지 라우트 추가
  - `GET /admin` HTML 서빙
  - `GET /` -> `/admin` 리다이렉트
- 정적 자원 마운트
  - `/admin/static` 경로에 JS/CSS/HTML 연결
- 대시보드 UI 3종 파일 추가
  - `src/api/static/admin/index.html`
  - `src/api/static/admin/styles.css`
  - `src/api/static/admin/app.js`

## 핵심 화면 구성
- 상단 상태바: 서버 상태, 모델명, 수동 새로고침
- KPI 카드: 총 이벤트, 응답 완료, MCP 사용 비율, 평균 신뢰도
- 이벤트 테이블: 최근 위험 이벤트 목록
- 상세 패널: 운영자 대응 문장, Jetson TTS, 참고 매뉴얼

## 데이터 연동
- 폴링으로 `GET /health`, `GET /events/recent` 주기 조회
- 이벤트 클릭 시 상세 응답 바인딩

## 확인 포인트
- `http://127.0.0.1:8000/admin` 접속 가능
- 이벤트 유입 시 테이블/KPI/상세 패널이 동기화됨
